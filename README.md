# Novvr Technologies

The official website of **Novvr Technologies**, built with **Next.js App Router**, **TypeScript**, **Tailwind CSS**, and **Supabase**.

The website is designed to be fast, SEO-first, responsive, and conversion-focused, showcasing Novvr's expertise in Web3, AI, Full-Stack Engineering, and Product Development.

---

## Development

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
